Zetflix

Link to github repo https://github.com/bushbass/interview/tree/master/Frontend/bushbass_frontend

Link to demo https://zetflix.netlify.app/

A full-stack MERN app with auth.

Pulls movie data from a public API. Users can create an account to login and create a list of favorite movies. Uses JWT for auth. The backend is built in Node/Express with MongoDB.

Backend Repo - https://github.com/bushbass/movie-list-backend

Skills: HTML | CSS | JavaScript | APIs | React | Node/Express | MongoDB | JWT - Jason Web Tokens | Auth

Hosted on Netlify
